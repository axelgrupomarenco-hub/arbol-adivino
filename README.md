# Arbol Adivino

Juego de preguntas de Si o No creado en Python con Programacion Orientada a Objetos,
CustomTkinter y archivos JSON.

## Ejecucion

Instalar dependencia:

```bash
pip install -r requirements.txt
```

Ejecutar:

```bash
python main.py
```

## Estructura

- `main.py`: punto de entrada.
- `clases/`: clases del arbol, nodos, archivos e interfaz.
- `datos/`: archivos JSON de ejemplo.
- `documentacion/diseno_poo.txt`: documento breve de diseno orientado a objetos.
